Description:

This program creates a PDF file for printing on a matrix of sticky labels. Input data must be in TSV format (like when copied from a spreadsheet). Indices select which columns will be used to populate each label. The fixed label format is the following:

[DATA_1]
[DATA_2] [DATA_3]
[DATA_4]
